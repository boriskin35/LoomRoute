# LoomRoute: Агентский контекстный каркас

Легковесный фреймворк для управления контекстом и маршрутизации ИИ-агентов, разработанный специально для терминальных кодеров (**Claude Code**, **Opencode**, **Antigravity 2.0**, **Qwen CLI** и **Mimo CLI**). Он организует ваше рабочее пространство в изолированные среды (например, `work` и `individual`), что позволяет сохранять контекст ИИ чистым, снижает расход токенов и внедряет структурированные рабочие процессы.

> [!WARNING]
> **Совместимость с IDE:** Этот фреймворк разработан исключительно для консольных CLI-агентов, которые автоматически считывают `CLAUDE.md` при старте. Он **несовместим** с IDE вроде Cursor или Windsurf «из коробки», так как они не поддерживают динамическое переключение локальных правил в процессе чат-сессии.

## 💡 Bilingual Repository / Двуязычный репозиторий

Этот репозиторий содержит двуязычную документацию (на английском и русском языках). Вы можете безопасно удалить файлы на ненужном вам языке:
- Чтобы оставить **только английский**: удалите все файлы `*.RU.md`.
- Чтобы оставить **только русский**: удалите английские `.md` файлы, а файлы `*.RU.md` переименуйте на их место (например, переименуйте `README.RU.md` в `README.md`).

## Быстрый старт

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/boriskin35/LoomRoute.git
   ```
2. Перейдите в папку проекта:
   ```bash
   cd LoomRoute
   ```
3. Запустите консольного ИИ-агента (например, команду `claude` или `qwen`).
4. Введите команду `/help`, чтобы увидеть список доступных команд (или просто напишите `help`, если ваша консоль или IDE перехватывает символ `/`).
5. Попробуйте создать новую среду, написав агенту: *"Создай новую среду с названием learn"* или напишите `help` для вывода опций.

## Структура рабочей области

- `CLAUDE.md`: Корневая конституция (ИИ-роль роутера).
- `registry.md`: Глобальная таблица маршрутизации сред.
- `skills/`: Глобальные навыки агента (на английском языке).
- `templates/`: Шаблоны документов (брифы, чек-листы).
- `work/`: Рабочая среда (код, веб-приложения, деплой).
- `individual/`: Персональная среда (Второй мозг, дневник, идеи).
